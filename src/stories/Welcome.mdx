A modern, opinionated chart components library built on **Recharts** and **Tailwind CSS v4** for React-based projects.

---

## Installation

```bash
npm install eglador-ui-react-chart
```

## Setup

Add the following to your global stylesheet (e.g. `app/globals.css`):

```css
@import "tailwindcss";
@source "../node_modules/eglador-ui-react-chart";
```

## Quick Start

```tsx
import { LineChart, BarChart, PieChart } from "eglador-ui-react-chart";
```

## Features

- **Config-driven API** — `data`, `xKey`, `series` props with minimal boilerplate
- **TypeScript-first** — full type safety on Recharts options
- **Tailwind v4 native** — default palette, optional per-series override
- **Responsive by default** — `responsive={false}` for fixed sizing
- **Full Recharts power** — `margin`, `barCategoryGap`, `reverseStackOrder`, `startAngle`/`endAngle` and more
- **Interactive Storybook** — edit `data` and `series` JSON live from the Controls panel
