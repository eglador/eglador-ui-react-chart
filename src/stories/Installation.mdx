# Installation

Package installation, peer dependencies, and Tailwind v4 integration.

---

## 1. Install the package

```bash
npm install eglador-ui-react-chart
```

```bash
yarn add eglador-ui-react-chart
```

```bash
pnpm add eglador-ui-react-chart
```

## 3. Add `@source` for Tailwind v4

Tailwind needs to scan classes inside the package. Add the `@source` directive to your global stylesheet:

```css
@import "tailwindcss";

@source "../node_modules/eglador-ui-react-chart";
```

The `@source` path varies based on your CSS file location:
