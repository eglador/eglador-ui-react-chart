# Getting Started

Build your first chart in a few lines.

> If you haven't installed the package yet, see the **Installation** page.

---

## 1. Import the component

```tsx
import { LineChart } from "eglador-ui-react-chart";
```

Other charts:

```tsx
import {
  LineChart,
  BarChart,
  AreaChart,
  PieChart,
  DoughnutChart,
  RadarChart,
  RadialBarChart,
  TreemapChart,
} from "eglador-ui-react-chart";
```

---

## 2. Prepare your data

Charts expect an array of objects with a category field (X axis) and numeric value fields:

```tsx
const data = [
  { month: "Jan", revenue: 4200 },
  { month: "Feb", revenue: 3100 },
  { month: "Mar", revenue: 5800 },
  { month: "Apr", revenue: 4900 },
  { month: "May", revenue: 6300 },
  { month: "Jun", revenue: 7200 },
];
```

---

## 3. Define series

Specify which key to plot, its label, and color:

```tsx
const series = [
  { key: "revenue", label: "Revenue", color: "#3b82f6" },
];
```

`color` is optional — defaults to the Tailwind palette if omitted.

---

## 4. Render

```tsx
import { LineChart } from "eglador-ui-react-chart";

export function MyChart() {
  return (
    <LineChart
      data={data}
      xKey="month"
      series={series}
      showGrid
      showTooltip
      showLegend
    />
  );
}
```

## Next Steps

- Pick a chart from **Charts/** in the sidebar → tweak props live in the Controls panel
- Click **Show code** in any story → copy-paste-ready usage examples
- Add multiple items to `series` for multi-line / multi-bar charts
- Override colors per item with `color: "#hex"`, or rely on the default palette

---

## Multiple series example

```tsx
<LineChart
  data={data}
  xKey="month"
  series={[
    { key: "revenue", label: "Revenue", color: "#3b82f6" },
    { key: "expenses", label: "Expenses", color: "#ef4444" },
    { key: "profit", label: "Profit", color: "#10b981" },
  ]}
  showGrid
  showTooltip
  showLegend
/>
```
